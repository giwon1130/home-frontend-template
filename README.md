# Home Frontend Template

React + Vite + TypeScript starter template for personal frontend projects.

## Stack
- React 18
- TypeScript 5
- Vite 7
- Vitest + Testing Library

## Quick Start
```bash
cp .env.example .env
pnpm install
pnpm dev
```

## Commands
```bash
pnpm dev
pnpm test
pnpm build
pnpm preview
```

## Project Structure
```
src
├── App.tsx
├── main.tsx
├── styles.css
├── components/
└── pages/
```
